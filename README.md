# BGMIcards Exchange Portal

A community card exchange portal for r/BGMIcards subreddit.

## Features
- Exchange Cards — List your cards for trade
- Find Cards — Search for cards you need
- Donate Cards — Donate extra cards to the community
- Stock — Check available card listings
- Trade Rules — Community guidelines

## Tech Stack
- React + Vite
- Supabase (database)
- Vercel (hosting)
